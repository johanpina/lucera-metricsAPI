Lucera Metrics API

API REST de solo lectura que alimenta el dashboard del cliente con datos reales del chatbot pediátrico. Rápida de conectar: cada respuesta ya trae la forma exacta de tus tipos.

Base URL
https://lucera-metrics-877493497677.us-central1.run.app
Auth
JWT · Bearer
Formato
JSON · UTF-8 · CORS abierto
💡 Filosofía

Los nombres de campos coinciden 1:1 con los tipos de tu src/lib/mockData.ts. Migrar es reemplazar import { acudientes } from "@/lib/mockData" por una llamada fetch — tus componentes no cambian.

Autenticación

La API usa JSON Web Tokens. El dashboard hace login una vez, recibe un token firmado y lo envía en Authorization en cada petición. El token expira a las 12 h; al caducar, la API responde 401 y hay que volver a loguear.

Conectar el dashboard (React)

Reemplaza los imports de mockData por este cliente. Guarda el token y adjúntalo en cada llamada.

src/lib/Copiar
const BASE = import.meta.env.VITE_API_URL ??
  "https://lucera-metrics-877493497677.us-central1.run.app";

export async function login(email: string, password: string) {
  const r = await fetch(`${BASE}/auth/login`, {
    method: "POST", headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ email, password }),
  });
  if (!r.ok) throw new Error("Correo o contraseña incorrectos");
  const { token, user } = await r.json();
  localStorage.setItem("lucera_token", token);
  return user;   // { email, nombre, rol } → guárdalo en tu AuthContext
}

export async function api<T>(path: string): Promise<T> {
  const token = localStorage.getItem("lucera_token");
  const r = await fetch(`${BASE}${path}`, { headers: { Authorization: `Bearer ${token}` } });
  if (r.status === 401) { localStorage.removeItem("lucera_token");
    throw new Error("Tu sesión expiró, vuelve a iniciar"); }
  if (!r.ok) throw new Error(`Error ${r.status}`);
  return r.json();
}
import { useQuery } from "@tanstack/react-query";
import { api } from "@/lib/api";
import type { Acudiente } from "@/lib/mockData";  // reutiliza tus tipos

export function useAcudientes() {
  return useQuery({
    queryKey: ["acudientes"],
    queryFn: () => api<Acudiente[]>("/api/acudientes"),
  });
}

Como los campos coinciden con mockData.ts, basta cambiar el import por useAcudientes(). Tus componentes (StatCard, TriageBadge, tablas) quedan igual.
